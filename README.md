
# Kirby - Helpers

My custom PHP & JS helpers for [Kirby](https://getkirby.com/).

# Install

- Git submodule: `git submodule add https://github.com/Daandelange/kirby-helpers.git ./site/plugins/daans-helpers`
- Composer (PHP classes and Kirby plugin): `composer install daandelange/helpers`
- NPM (vue js modules) : `npm install git+https://github.com/Daandelange/kirby-helpers.git`
